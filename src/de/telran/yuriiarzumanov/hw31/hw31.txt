Что такое Spring Framework?
Spring Framework - это фреймворк для разработки приложений на языке Java. Он предоставляет комплексный набор
инструментов для создания различных типов приложений, включая веб-приложения, корпоративные приложения и микросервисы.

В чем различия между Spring Framework и Spring Boot?
Spring Framework: Общий фреймворк для создания приложений, требует более явной конфигурации.
Spring Boot: Подпроект Spring, упрощающий создание структуры и конфигурации приложений, предоставляет множество
конвенций для уменьшения необходимости явной настройки.

Какие методы есть в REST API?
GET: Получение данных.
POST: Отправка данных для создания ресурса.
PUT: Обновление данных ресурса.
DELETE: Удаление ресурса.
PATCH: Частичное обновление ресурса.

Для чего используют GET, а для чего POST?
GET: Используется для запроса данных от сервера. Данные передаются в URL.
POST: Используется для отправки данных на сервер. Данные передаются в теле запроса.

Какие ответы есть в серверных приложениях?
HTTP Status Codes: Информируют клиента о статусе выполнения запроса (например, 200 OK, 404 Not Found).
Тело ответа: Данные, отправляемые обратно клиенту (например, JSON, HTML).
Заголовки ответа: Дополнительная информация о ответе.

Можно ли собрать jar файл в Spring Boot?
в Spring Boot можно собрать JAR файл.С помощью инструмента сборки, такого как Maven или Gradle, проект Spring Boot
компилируется и упаковывается в исполняемый JAR файл
