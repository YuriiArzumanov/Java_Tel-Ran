Что такое параллельные вычисления?
Это метод обработки информации, при котором задача разделяется на подзадачи, которые выполняются одновременно.
Это может происходить на множестве процессоров или ядер.

В чем преимущество параллельных вычислений?
Ускорение обработки данных.
Эффективное использование многозадачности.
Решение более сложных задач.

Что такое поток и процесс? В чем их разница?
Процесс: Это независимый исполняемый экземпляр программы.
Поток: Это наименьшая единица обработки, которая может быть запущена и исполняется внутри процесса.

Как можно создать поток?

Thread thread = new Thread(new Runnable() {
    public void run() {
        // Код в потоке
    }
});
thread.start();


Какие преимущества и недостатки создания потока каждым из способов?
Преимущества:
Увеличение параллельности и производительности.
Эффективное использование многозадачности.
Недостатки:
Потенциальные проблемы с синхронизацией данных между потоками.
Усложнение управления ресурсами.

В чем недостатки при использовании многопоточности?
Проблемы с безопасностью данных при общем доступе.
Сложности в отладке из-за возможных состояний гонки.

Выполняется ли поток параллельно? Как происходит работа если 1000 потоков, а процессор только 1?
потоки выполняются параллельно в пределах доступных ресурсов.
выполнение потоков происходит последовательно чередованием между ними,может возникнуть ухудшение производительности
из-за переключения контекста между потоками.
